# OCR 오인식 백과사전

한글·영문 형태 유사로 인한 착시 오인식. **글자(자모)·음절·단어** 단위 혼동 패턴.

---

## 1. 한글 자모 → Latin 오인식

**자음**

**ㄴ** → L, r, n (직선+굴곡)  
**ㄹ** → L, r, ll (이중 직선)  
**ㅇ** → O, o, 0 (원형)  
**ㅁ** → M, m (사각형)  
**ㅅ** → s, S, c (곡선)  
**ㄱ** → r, L, 7 (꺾임)  
**ㅈ** → J, j (위에 가로선)  
**ㅊ** → C, c (곡선+가로선)  
**ㅋ** → K, k (세로+대각선)  
**ㅌ** → T, t (가로+세로)  
**ㅍ** → P, p (세로+가로)  
**ㅎ** → H, h (가로 두 줄)  
**ㅂ** → B, b (세로+가로)  
**ㄷ** → d, D (곡선+가로)  

**모음**

**ㅣ** → l, I, 1 (세로 직선)  
**ㅡ** → -, 一 (가로 직선)  
**ㅗ** → T, t (세로+가로)  
**ㅜ** → u, U (굴곡)  
**ㅏ** → a, A (세로+가로)  
**ㅓ** → e, E (굴곡)  
**ㅐ** → ae, ai  
**ㅔ** → e, E  

---

## 2. Latin → 한글 자모 오인식

**L** → ㄴ, ㄹ, ㅣ  
**l** → ㅣ, ㄴ, 1  
**I** → ㅣ, 1, l  
**O** → ㅇ, 0  
**o** → ㅇ  
**0** → ㅇ, O  
**M** → ㅁ  
**m** → ㅁ  
**s** → ㅅ  
**S** → ㅅ  
**r** → ㄱ, ㄴ, ㄹ  
**n** → ㄴ  
**J** → ㅈ  
**C** → ㅊ, ㅅ  
**K** → ㅋ  
**T** → ㅌ, ㅗ  
**t** → ㅌ, ㅗ  
**P** → ㅍ  
**p** → ㅍ  
**H** → ㅎ  
**h** → ㅎ  
**B** → ㅂ  
**b** → ㅂ  
**D** → ㄷ  
**d** → ㄷ  
**1** → ㅣ, l, I  
**-** → ㅡ  
**!** → ㅣ, 1 (PII에서 I→!)  
**@** → a, ㅏ (Token@ 등)  

---

## 3. 한글 음절(완성형) 오인식

**ㅇ↔ㅁ** (받침·초성): 안↔만, 인↔민, 억↔먹, 언↔먼  
**ㅈ↔ㅅ**: 조↔소, 정↔청  
**전↔천**: 전반↔천반  
**월↔뭘**: 월↔뭘 (ㅗㅓ↔ㅜㅓ)  
**성↔섬**: 성↔섬 (받침 ㅇ↔ㅁ)  
**중↔줌**: 중↔줌 (ㅜㅇ↔ㅜㅁ)  
**방↔밤**: 방↔밤 (ㅇ↔ㅁ)  
**알↔말**: 알↔말 (ㅇ↔ㅁ)  
**사↔샤**: 사↔샤 (의사결정↔의샤결정)  
**감↔같**: 감↔같 (ㅁ↔ㅌ)  
**도↔노**: 도↔노 (경우에도↔경우메도)  

---

## 4. 숫자·기호 오인식

**P** → 0 (PII→0!!)  
**I** → ! (PII→0!!)  
**2** → 이, ㅣ  
**5** → 오, ㅗ  
**1** → ㅣ, l, I  
**0** → ㅇ, O  
**@** → a  
**|** → I, ㅣ  

---

## 5. 자주 혼동되는 조합 (요약)

**ㅇ vs ㅁ**: 안↔만, 인↔민, 억↔먹, 언↔먼, 중↔줌, 방↔밤, 알↔말  
**ㅈ vs ㅅ**: 조↔소, 조정↔소정, 위변조↔위변소  
**ㄴ vs ㄹ**: ㄴㄴ↔LL (LLM 오인식)  
**ㅣ vs l vs I vs 1**: 세로 직선 4종  
**ㅇ vs O vs 0**: 원형 3종  
**전 vs 천**: 전반↔천반메  

---

## 6. 활용

- **전처리**: ㅇ/ㅁ, ㅈ/ㅅ 등 혼동 자모 경계 선명화(sharpen, morphology)
- **후처리**: `postprocess.OCR_TYPO_MAP`에 음절·단어 단위 매핑 적용. 본 문서 패턴 참고
- **PSM·이진화**: 형태가 뭉개지면 혼동 증가 → 전처리로 자모 분리 유지
