# OCR 오인식 백과사전

한글·영문 형태 유사로 인한 착시 오인식 쌍. 전문 용어가 아닌 **글자 단위** 혼동 패턴.

---

## 한글 자모 → Latin 오인식

**ㄴ** → L, r, n (직선+굴곡)  
**ㄹ** → L, r, ll (이중 직선)  
**ㅇ** → O, o, 0 (원형)  
**ㅁ** → M, m (사각형)  
**ㅅ** → s, S, c (곡선)  
**ㄱ** → r, L, 7 (꺾임)  
**ㅈ** → J, j (위에 가로선)  
**ㅊ** → C, c (곡선+가로선)  
**ㅋ** → K, k (세로+대각선)  
**ㅌ** → T, t (가로+세로)  
**ㅍ** → P, p (세로+가로)  
**ㅎ** → H, h (가로 두 줄)  
**ㅂ** → B, b (세로+가로)  
**ㄷ** → d, D (곡선+가로)  
**ㅣ** → l, I, 1 (세로 직선)  
**ㅡ** → -, 一 (가로 직선)  
**ㅗ** → T, t (세로+가로)  
**ㅜ** → u, U (굴곡)  
**ㅏ** → a, A (세로+가로)  
**ㅓ** → e, E (굴곡)  
**ㅐ** → ae, ai  
**ㅔ** → e, E

**L** → ㄴ, ㄹ, ㅣ  
**l** → ㅣ, ㄴ, 1  
**I** → ㅣ, 1, l  
**O** → ㅇ, 0  
**o** → ㅇ  
**0** → ㅇ, O  
**M** → ㅁ  
**m** → ㅁ  
**s** → ㅅ  
**S** → ㅅ  
**r** → ㄱ, ㄴ, ㄹ  
**n** → ㄴ  
**J** → ㅈ  
**C** → ㅊ, ㅅ  
**K** → ㅋ  
**T** → ㅌ, ㅗ  
**t** → ㅌ, ㅗ  
**P** → ㅍ  
**p** → ㅍ  
**H** → ㅎ  
**h** → ㅎ  
**B** → ㅂ

**D** → ㄷ  
**d** → ㄷ

---

## 자주 혼동되는 조합

**ㅇ vs ㅁ** (받침·초성): 안↔만, 인↔민, 억↔먹  
**ㅈ vs ㅅ**: 조↔소, 조정↔소정  
**ㄴ vs ㄹ**: ㄴㄴ↔LL (LLM 오인식)  
**ㅣ vs l vs I vs 1**: 세로 직선 4종  
**ㅇ vs O vs 0**: 원형 3종  

---

## 활용

- 전처리: ㅇ/ㅁ, ㅈ/ㅅ 등 혼동 자모 경계 선명화(sharpen, morphology)
- 후처리: 문맥 기반 치환 시 참고 (단, 특정 용어 매몰 주의)
- PSM·이진화: 형태가 뭉개지면 혼동 증가 → 전처리로 자모 분리 유지
