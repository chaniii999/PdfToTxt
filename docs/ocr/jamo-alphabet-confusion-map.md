# 자모-알파벳 혼동 사전 (문자 단위)

ocr-misrecognition-encyclopedia 기반. **문자 단위**로 적용. 인접 문자가 ASCII일 때만 자모→알파벳 치환.

---

## 1. JAMO_TO_ALPHA (postprocess 적용)

| 자모 | → | 알파벳 |
|------|---|--------|
| ㄴ | → | L |
| ㄹ | → | L |
| ㅁ | → | M |
| ㅇ | → | O |
| ㅅ | → | S |
| ㅂ | → | B |
| ㅎ | → | H |
| ㅈ | → | J |
| ㅊ | → | C |
| ㅋ | → | K |
| ㅌ | → | T |
| ㅍ | → | P |
| ㄷ | → | D |
| ㄱ | → | r |
| ㅣ | → | I |
| ㅓ | → | e |
| ㅏ | → | a |
| ㅗ | → | o |
| ㅜ | → | u |
| ㅡ | → | - |

---

## 2. 적용 조건

- **인접 문자가 ASCII(A-Za-z0-9)일 때만** 치환
- 예: "ㄴㄴM" → 2번째 ㄴ(옆에 M) → L → "ㄴLM" → 1번째 ㄴ(옆에 L) → L → "LLM"
- 다중 패스로 연쇄 치환
